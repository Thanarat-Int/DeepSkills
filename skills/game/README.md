# Game Studio skills

A focused studio for building games of any genre — standby specialists that fire only on game work.

- **[game-design](./game-design/SKILL.md)** — Design the fun: pillars, core loop, interacting systems, progression, economy, levels, and balance.
- **[game-narrative](./game-narrative/SKILL.md)** — Craft the story and world so it serves play: premise/theme, worldbuilding, characters, dialogue, quests, and meaningful branching.
- **[game-dev](./game-dev/SKILL.md)** — Engineer it for real-time: game loop/timestep, state/ECS, scenes, asset pipeline, frame budget, and multiplayer netcode/anti-cheat.
