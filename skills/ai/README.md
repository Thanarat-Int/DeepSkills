# AI / ML Lab skills

A world-class AI engineering team, split by **discipline** (not domain) so it handles any model or task — CV (OCR, detection), NLP, tabular, time-series, GenAI. Standby specialists that fire only on AI/ML work.

- **[data-engineering](./data-engineering/SKILL.md)** — Build the data foundation: collect, clean, label, transform, version, and split data without leakage.
- **[ml-modeling](./ml-modeling/SKILL.md)** — Train/fine-tune/evaluate models for any task: baseline first, transfer learning by default, honest evaluation on the right metric.
- **[mlops](./mlops/SKILL.md)** — Get models into production and keep them working: serve, version (model+data+code), monitor drift, automate retraining and rollback.
- **[llm-engineering](./llm-engineering/SKILL.md)** — Build reliable systems on LLMs: prompt → RAG → fine-tune, tool-use/agents, rigorous evals and guardrails.
